# 🎬 Netflix Data Analysis

Netflix content analysis using Python, Pandas, Matplotlib and Streamlit.

## 📌 About
This project analyzes Netflix's content library to find interesting patterns 
and trends using data analysis and visualization techniques.

## 📊 What I Analyzed
- Movies vs TV Shows distribution
- Top 10 most popular genres
- Top 10 countries producing content
- Content added per year trend
- Top 10 content ratings

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Streamlit
- Jupyter Notebook

## 📁 Files
- `Netflix_data_analysis.ipynb` → Main analysis notebook
- `app.py` → Streamlit web app
- `netflix_data.csv` → Dataset

## 🔍 Key Findings
- Netflix has significantly more Movies than TV Shows
- USA is the top content producing country
- Comedies and Dramas are the most popular genres
- Netflix content grew rapidly between 2015-2020
